## Problems

### Q1 — Decision Tree on Titanic Dataset
Built a Decision Tree classifier from scratch without scikit-learn. Implements:
- Entropy and Information Gain computation
- Recursive best-split selection
- Tree traversal for prediction
- Evaluation: Confusion Matrix, F1-Score

**Dataset:** Titanic survival prediction

### Q2 — Simple Linear Regression: TV Marketing → Sales
Linear regression with standardization and gradient descent optimization.

### Q3 — Multiple Linear Regression: Boston Housing
13-feature regression predicting MEDV (median house values) using gradient descent.
