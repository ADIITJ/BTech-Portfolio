# Deep Learning — NLP Assignments

**Subject:** Deep Learning (Y3)
**Institution:** IIT Jodhpur, Dept. of CSE & AI

## Problem
Coursework assignments covering Natural Language Understanding (NLU) and NLP tasks using deep learning frameworks.

## Tech Stack
- Python, PyTorch / TensorFlow
- Jupyter Notebooks
- NLTK, HuggingFace

## Contents
- `train.py` — model training pipeline
- `nlu.py` — NLU task implementation
- `*.ipynb` — per-assignment notebooks (DLOPS, NLP tasks)

## How to Run
```bash
pip install -r requirements.txt
jupyter notebook
```
