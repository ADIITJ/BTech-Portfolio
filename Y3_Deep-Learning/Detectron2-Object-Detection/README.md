# Deep Learning — Object Detection with Detectron2

**Subject:** Deep Learning (Y3)
**Institution:** IIT Jodhpur, Dept. of CSE & AI

## Problem
Object detection implementation using Facebook Research's Detectron2 framework. Explores pre-trained model inference and result visualization on custom input images.

## Tech Stack
- Python
- Detectron2 (Facebook Research)
- PyTorch, OpenCV

## Contents
- `detectron2/` — framework source / config
- `image/` — input images
- `output/`, `output1/` — detection results

## How to Run
```bash
pip install detectron2
python demo.py --input image/ --output output/
```
