# Machine Learning for Economics — Assignment 2: Credit Risk Analysis

**Subject:** Machine Learning for Economics (Y4)
**Institution:** IIT Jodhpur, Dept. of CSE & AI

## Problem
Binary classification for credit risk assessment. Predicts loan default probability using borrower features. Includes model comparison, feature importance, and performance visualization.

## Tech Stack
- Python, pandas, scikit-learn

## Key Files
- `credit_risk.ipynb` — analysis notebook
- `credit_risk.csv` — dataset
- `result_charts/` — performance visualizations
- `MLE_2.pdf` — assignment specification

## Run
```bash
pip install pandas scikit-learn jupyter
jupyter notebook credit_risk.ipynb
```
