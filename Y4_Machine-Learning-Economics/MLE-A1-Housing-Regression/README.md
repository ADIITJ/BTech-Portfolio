# Machine Learning for Economics — Assignment 1: Housing Regression Analysis

**Subject:** Machine Learning for Economics (Y4)
**Institution:** IIT Jodhpur, Dept. of CSE & AI

## Problem
Exploratory data analysis and regression modeling on housing price data. Covers feature engineering, correlation analysis, and supervised regression with performance evaluation.

## Tech Stack
- Python, pandas, matplotlib, scikit-learn

## Key Files
- `housing_analysis.ipynb` — interactive analysis notebook
- `housing_analysis.py` — standalone script
- `*.csv` — dataset
- `Assignment 1.pdf` — problem specification

## Run
```bash
pip install pandas matplotlib scikit-learn jupyter
python housing_analysis.py
```
