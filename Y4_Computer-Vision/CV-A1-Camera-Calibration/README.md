# Computer Vision — Assignment 1: Camera Calibration

**Subject:** Computer Vision (Y4)
**Institution:** IIT Jodhpur, Dept. of CSE & AI

## Problem
Estimate intrinsic and extrinsic camera parameters using checkerboard pattern detection. Implements the standard Zhang (2000) calibration pipeline: corner detection → homography estimation → parameter extraction.

## Tech Stack
- Python, OpenCV, NumPy

## Key Files
- `q1.py` — checkerboard corner detection and calibration
- `q2.py` — reprojection error analysis
- `run.sh` — execution script
- `CV_1.pdf` — assignment specification

## Run
```bash
bash run.sh
# or
python q1.py
```
