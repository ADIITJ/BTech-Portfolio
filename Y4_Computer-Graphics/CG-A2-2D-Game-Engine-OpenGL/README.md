# Computer Graphics — Assignment 2: 2D Game Engine (OpenGL + ImGUI)

**Subject:** Computer Graphics (Y4)
**Institution:** IIT Jodhpur, Dept. of CSE & AI

## Problem
Build an interactive 2D game engine using OpenGL for rendering and ImGUI for immediate-mode UI. Implements a game loop, sprite rendering, collision detection, and real-time parameter tweaking via GUI panels.

## Tech Stack
- Python
- OpenGL (PyOpenGL)
- ImGUI (pyimgui)
- GLFW

## Key Files
- `main.py` — engine entry point and game loop
- `game.py` — game logic
- `utils/` — rendering utilities
- `assets/` — sprites and resources
- `B22AI045-A2.pdf` — assignment report
- `ReadMe.pdf` — setup instructions

## Run
```bash
pip install PyOpenGL pyimgui glfw
python main.py
```
