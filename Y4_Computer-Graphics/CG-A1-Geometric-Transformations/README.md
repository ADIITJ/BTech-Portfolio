# Computer Graphics — Assignment 1: Geometric Transformations

**Subject:** Computer Graphics (Y4)
**Institution:** IIT Jodhpur, Dept. of CSE & AI

## Problem
Implement 2D/3D geometric transformations (translation, rotation, scaling, projection) and camera model fundamentals. Output images demonstrate transformation effects on geometric primitives.

## Tech Stack
- Python, OpenCV / image processing libraries, NumPy

## Key Files
- `b22ai045_1.py`, `b22ai045_2.py` — transformation implementations
- `*.png` — output visualizations
