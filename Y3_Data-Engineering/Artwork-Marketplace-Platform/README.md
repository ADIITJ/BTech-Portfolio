# Data Engineering — Artwork Marketplace Platform

**Subject:** Data Engineering (Y3)
**Institution:** IIT Jodhpur, Dept. of CSE & AI

## Problem
Full-stack art marketplace web application with user authentication, artwork browsing, and data management. Includes synthetic data generation for testing and a de-hashing utility for credential management.

## Tech Stack
- Python (Streamlit / Flask)
- SQLite / MongoDB
- Web templating

## Key Files
- `app.py` — application entry point
- `db.py` — database layer
- `generate_data.py` — synthetic data generation
- `dehashing.py` — credential utility

## Run
```bash
pip install -r requirements.txt
python app.py
```
